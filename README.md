# SocialProfileKit

A UIKit-based demo app that simulates a simple social media Feed and Profile screen.

## Features
- Tab bar navigation (Feed / Profile)
- Feed: vertical list layout using UICollectionView
- Profile: custom header + grid posts layout
- Reusable PostCell across screens
- Story horizontal scroll
- Empty state handling

## Tech Stack
- UIKit
- UICollectionView + UICollectionViewFlowLayout

## Project Structure
- App: AppDelegate, SceneDelegate
- Scenes: Feed, Profile
- Views: Cells, Headers
- Models

## How to Run
Open `SocialProfileKit.xcodeproj` and run on iOS Simulator.
